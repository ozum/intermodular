assign

  • assign(path: string | string[] | undefined, data: Record<string, any>, conditions?: ModifyCondition): this
    • Defined in data-file.ts:220

    Merges all keys and values of data shallowly into path of file data. If a portion of path doesn't exist, it's created. Differnet to object assign, keys may be merged conditionally such as ifExists or ifNotExists.

    Example

     
    const packageJson = targetModule.getDataFileSync("package.json"); // `DataFile` instance
packageJson.assign("scripts", { build: "tsc", test: "jest", }, { ifNotExists: true });

    Example

     
    const packageJson = targetModule.getDataFileSync("package.json"); // `DataFile` instance
packageJson.assign("scripts", { build: "tsc", test: "jest", }, { ifNotExists: true });

orderKeys

  • orderKeys(keys?: keyof T[]): this
    • Defined in data-file.ts:252

    When keys/values added which are previously does not exist, they are added to the end of the file during file write. This method allows reordering of the keys. keys are placed at the beginning in given order whereas remaining keys of the object comes in their order of position.

    Example

     
    const packageJson = targetModule.getDataFileSync("package.json"); // `DataFile` instance
packageJson.orderKeys(["name", "version", "description", "keywords", "scripts"]); // Other keys come after.

    Example

     
    const packageJson = targetModule.getDataFileSync("package.json"); // `DataFile` instance
packageJson.orderKeys(["name", "version", "description", "keywords", "scripts"]); // Other keys come after.

    Parameters

    • Default value keys: keyof T[] = Object.keys(this.data).sort()

      are ordered keys to appear ath beginning of file when saved.

    Returns this
